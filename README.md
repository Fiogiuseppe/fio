# FIO — fiogiuseppe.com

Public portfolio site for Giuseppe Fioretti. Built with Next.js, deployed on Vercel at [fiogiuseppe.vercel.app](https://fiogiuseppe.vercel.app).

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
